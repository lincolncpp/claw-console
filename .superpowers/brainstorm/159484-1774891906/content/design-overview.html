Component Breakdown

Panel Shell

Drag handle (top), header bar with agent/session/status info, close button. Remembers height in localStorage.

Chat Log

Monospace messages with role labels. Auto-scrolls to bottom. Supports markdown in assistant replies.

Tool Cards

Dashboard-style cards with collapse/expand. Shows tool name, status badge, duration. Args/results hidden by default.

Input Area

Single-line input with prompt character. Enter to send. Shows streaming cursor when assistant is responding.
